# Changelog

All notable OscarUI changes are documented here. Releases follow semantic versioning.

## [1.0.0] - 2026-07-11

### Added

- Deterministic Swift/SwiftUI and Kotlin/Jetpack Compose generation from shared UI IR.
- Reusable components, native application metadata, navigation, lists, forms, and native action extension points.
- Optional Runtime Mode with portable bundles, integrity checks, caching, remote updates, and last-known-good fallback.
- iOS and Android development commands, screenshot parity checks, and visual regression review.
- Installable `oscarui` CLI with clean-project initialization and package smoke testing.
